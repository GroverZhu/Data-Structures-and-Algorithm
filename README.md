# 运行项目

- 支持的平台：Linux, MacOS
- 软件：g++ 或者 LLVM, CMAKE
- 项目编译：在该文件夹下建立 `build` 文件夹，进入 `build` 文件夹运行 `cmake ..`，要在 debug 模式下运行命令为 `cmake -DCMAKE_BUILD_TYPE=DEBUG ..`, 使用 `make all` 编译所有代码，在 `build` 文件夹下有 `bin` 文件，里面为编译后的可执行文件
